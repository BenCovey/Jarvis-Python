Jarvis Python Voice Recognition

Project Description

Built in Python; Jarvis is a program that uses voice recognition to complete tasks for you on a Windows computer. Able to open control windows, open directories, open applications, make notes, set alarms, read and parse weather sites, and much more. All feedback from the program is via a speech function where Jarvis will actually talk to you.

Project Details

  • Voice Recognition
  • Speech synthesizer
  • Control of Windows
  • Hands free.
